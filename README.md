✨ **This workspace has been generated by [Nx, a Smart, fast and extensible build system.](https://nx.dev)** ✨

## Create environement

```sh
npx create-nx-workspace@latest
yarn add --dev @nx/next
yarn nx g @nx/next:app admin
```

## Running App

```sh
yarn nx serve admin
```
